### 11. 주요 구현 내용 - 게임 매니저 (1장)

```csharp

- Photon 서버 연결 및 방 생성
- 플레이어 입장/퇴장 처리
- 3명 플레이어 관리 (MAX_PLAYERS = 3)
```

**주요 기능:**
- `OnConnectedToMaster()`: 마스터 서버 연결
- `OnJoinedRoom()`: 방 입장 시 플레이어 ID 할당
- `GetLocalPlayerIndex()`: 현재 플레이어의 인덱스 반환
- `UpdateTurn()`: 턴 변경 시 플레이어 위치 이동

### 16. 주요 구현 내용 - 네트워크 동기화 (1장)
**Photon PUN2 활용:**
```csharp
// RPC (Remote Procedure Call)
[PunRPC]
void RPC_NextTurn(int nextPlayerIndex)
{
    currentPlayerTurn = nextPlayerIndex;
    NotifyAllPlayersOfTurnChange();
}

// 동기화 호출
photonView.RPC("RPC_NextTurn", RpcTarget.All, nextIndex);
```

**동기화 요소:**
- 턴 변경
- 버튼 클릭
- 점수 업데이트
- 플레이어 위치/회전

**네트워크 최적화:**
- RpcTarget.All / Others / MasterClient 적절히 사용
- Ownership Transfer로 오브젝트 소유권 관리

### 18. 개발 과정 및 이슈 해결 (1장)
**주요 개발 이슈:**

**1. VR 멀티플레이어 동기화 문제**
- **문제**: 플레이어 위치가 동기화되지 않음
- **해결**: PhotonTransformView 컴포넌트 추가 및 설정

**2. 턴제 시스템 구현**
- **문제**: 관전 플레이어의 카메라 동기화
- **해결**: Update()에서 활성 플레이어 HMD Transform 추적

**3. VR 인터랙션 이슈**
- **문제**: Ray Interactor가 버튼을 인식 못함
- **해결**: XRSimpleInteractable 컴포넌트 추가 및 Collider 설정

**4. 씬 전환 시 오브젝트 소실**
- **문제**: DontDestroyOnLoad 오브젝트 관리
- **해결**: 싱글톤 패턴으로 GameManager 유지
